# 龙头
**板块轮动：** 资金是“喜新厌旧”的。这个月炒“AI”，下个月炒“医药”，再下个月炒“白酒”。你不能“死守”一个板块。

**情绪周期：**

1. **冰点期：** 市场一片恐慌，股票大面积“跌停”，没人敢买。
2. **启动期：** 有“先锋”板块（比如“券商”）开始“试错”拉升，点燃一点火苗。
3. **发酵期：** 某个“主线”（比如“AI”）被确立，龙头股（领涨的）连续“涨停”。
4. **高潮期：** 所有人（包括你我）都“嗨”了，开始FOMO（害怕错过），连“垃圾股”都沾边就飞。**（这是最危险的时候）**
5. **退潮期：** 龙头股“熄火”倒下，资金“出逃”，市场“一地鸡毛”，又回到“冰点”。

**⚠️ 强风险警示：**

1. **这不是投资，这是投机：** 它不关心公司基本面，只关心“势”。
2. **极高风险：** 这种策略的收益和风险完全同源。你可能在几天内获利50%，也可能在几天内亏损50%。“龙头”一旦反转，下跌通常是连续跌停（stuck limit-down），你可能**连卖出的机会都没有**。
3. **反人性：** 它要求你在别人恐惧时买入（分歧），在别人疯狂时卖出（一致）。这需要钢铁般的纪律。

# 理论模块

### 阶段一：理念篇 —— 为什么会有龙头？

龙头战法的核心是A股的**“题材（Narrative）”**和**“情绪（Sentiment）”**。

1. **题材驱动：** 市场需要一个“故事”（如AI、对台、新质生产力）。
2. **资金共识：** 当一个新题材出现时，市场上的活跃资金（游资）会迅速寻找最正宗、最有辨识度的标的。
3. **形成龙头：** 第一个涨停、涨停封单最坚决的股票，会吸引所有人的目光。当它第二天继续涨停，就会被市场“加冕”为龙头。
4. **正反馈循环：** 龙头涨停 -> 吸引更多资金 -> 强化其龙头地位 -> 带动“小弟”（跟风股）上涨 -> 整个板块高潮 -> 龙头继续涨停。

### 阶段二：识别篇 —— 如何找到龙头？

龙头是在市场博弈中“打”出来的，而不是“选”出来的。你需要观察**盘面信号**。

- **1. 辨识度（看名字）：**
    - 这就是为什么“平潭发展”能成为龙头。它的名字（平潭）和题材（对台）直接挂钩。
    - 反例：一个叫“XX科技”的，但业务也涉及对台，它的辨识度就远不如“平潭发展”。
- **2. 率先涨停（领涨性）：**
    - 当一个题材（利好消息）在盘中发酵时，**第一个**涨停的股票最有可能是龙头。
    - 它定义了板块的高度。
- **3. 封单强度（“资金的决心”）：**
    - 观察涨停板上的封单量（排队买入的资金）。如果封单巨大（例如占流通盘的10%以上），说明市场高度看好，第二天溢价（高开）的概率极高。
- **4. 市场地位（“真龙” vs “跟风”）**
    - **龙头（总司令）：** 它涨停，小弟（跟风股）才敢涨。它开板（炸板），小弟会立刻跳水。
    - **跟风（小弟）：** 涨幅弱于龙头，涨停晚于龙头。
    - **卡位（新龙头）：** 当老龙头涨不动时（通常是3-5板后），板块里跳出另一只股票率先涨停，接替老龙头，这叫“龙头卡位”。

|**信号维度**|**最强龙头信号**|**较弱信号**|
|---|---|---|
|**消息面**|新闻具有**唯一性**、**重磅性**（如行业颠覆）|只是“蹭热点”，或“旧闻新炒”|
|**盘面**|**第一个**对消息做出反应，**率先**涨停|跟风涨停（“龙二”、“龙三”）|
|**分时图**|**垂直拉升**，大单扫货，攻击性强|“锯齿波”拉升，犹豫不决|
|**封板时间**|集合竞价“一字板”或上午10点前|下午涨停，尤其是尾盘|
|**封单质量**|**封单 / 流通市值 > 10%**（越高越好）|封单很小，或比例极低|
|**封板稳定**|全天未开板|盘中**多次炸板**（烂板）|

### 阶段三：周期篇 —— 龙头的“生老病死”

这是龙头战法**最核心的部分**。任何题材都有一个情绪周期，龙头是这个周期的“温度计”。

1. **启动期（分歧）：**
    - **信号：** 第1-2个涨停板（一板、二板）。
    - **市场心理：** “这是个啥？是新题材吗？” “是不是一日游？” 市场充满怀疑。
    - **盘面：** 可能会“烂板”（涨停被多次砸开再封上），成交量放大。
2. **加速期（一致）：**
    - **信号：** 第3-5个涨停板。
    - **市场心理：** “龙头确立了！” “再不买就没机会了！” 市场从怀疑转向一致看多。
    - **盘面：** 普遍“缩量涨停”（T字板或一字板），因为前一天买入的人惜售，外面的人买不到。
3. **高潮期（分歧转一致）：**
    - **信号：** 第5板以上。
    - **市场心理：** “疯了！还能涨！” 极度亢奋。
    - **盘面：** **“高位放巨量”**。龙头在涨停价附近剧烈震荡，成交量（换手率）创下天量。这是最危险的信号，说明早期获利的资金（主力）和新冲进来的资金（散户）在激烈交换筹码。
4. **衰退期（一致转分歧）：**
    - **信号：** 龙头在高位巨量震荡后，第二天无法涨停，或直接低开大跌。
    - **市场心理：** “要跑了！” “谁来接盘？” 恐慌开始。
    - **盘面：** **“跟风股”先于龙头大跌**（这就是你上一问提到的）。当“小弟”们开始跌停，说明板块情绪已经退潮，“总司令”（龙头）也即将补跌。

### 阶段四：策略篇 —— 如何“上车”与“下车”？

这是最难的实操部分，也是风险的来源。

“上车”的典型战术（高风险动作）：

- **打板（Hitting the Limit-Up）：** 在股票即将涨停或已经涨停的瞬间，排队买入。赌的是第二天的“高开溢价”。这是最主流的龙头战术，需要极快的手速和专用的交易通道。
- **半路（Half-Way）：** 在龙头启动当天，涨幅在3%-7%时，判断它今天能封住涨停，提前买入。
- **低吸（Buying the Dip）：** 参与龙头股的第一次回调。例如，在连续5个涨停后，出现了第一根阴线（下跌），赌它会“龙回头”（再次上涨）。

“下车”的离场信号（保命关键）：

1. **“小弟”先崩：** 如你所说，你的“海峡西岸”板块里，平潭发展还在涨，但厦门港务、漳州发展开始大跌了 -> **警惕信号**。
2. **龙头“炸板”：** 龙头涨停被巨量卖单砸开，并且当天无法再次封住。
3. **高位巨量阴线：** 龙头在高位放出天量，但收盘是绿的（下跌）。这是最明确的出货信号。
4. **开盘不及预期：** 龙头前一天涨停，但第二天开盘“低开”或“平开”，说明情绪已经转弱。

### 阶段五：纪律篇 —— 交易者的“自我修养”

龙头战法90%靠纪律和心态，10%才是技术。

1. **止损（Stop-Loss）：** 这是**铁的纪律**。如果买入后，走势不符合预期（例如，打板后第二天低开低走），必须**无条件卖出**。龙头战法绝不“补仓”，因为你买的是“势”，“势”不在了，就必须离场。
2. **仓位（Position Sizing）：** 绝不能全仓（All-in）。因为试错率很高，你可能要尝试3次才能抓到1次“真龙”，必须用小仓位去试错，抓到后再加仓。
3. **模式（Your Model）：** 专注一种战术。你是做“打板”的，就只做打板；你是做“低吸”的，就只做低吸。不要在不同模式间跳来跳去。

- 每天收盘后，交易所会公布当天“涨幅/跌幅/换手率”排名前几的股票。

如何用

- **一看“大佬”：** A股有很多“著名游资”（比如你可能听过的“赵老哥”、“章盟主”等）。他们有特定的“席位”。如果某只股票涨停，龙虎榜显示是“赵老哥”买了1个亿，那市场人气第二天就会被点燃，大家会“抄作业”跟着买。
- **二看“机构”：** 如果买入席位是“机构专用”，那分量就重了。这说明“公募基金”这种“大部队”进场了，他们一般不玩“一日游”，说明**中线看好**。
- **三看“接力”：** 如果“买一”的大佬今天买了，明天接着买（叫“锁仓”），说明看好；如果他今天买了，明天开盘就“砸盘”卖出，那说明是“一日游”的“渣男”行为。

## 资金流向

**央行的“水龙头” (宏观)：**

- **看“降准”和“降息”。**
- **降准 (RRR Cut)：** 银行能放出来的钱多了。
- **降息 (Interest Rate Cut)：** 大家借钱的成本低了，存钱的利息少了。
- 这俩都是“放水”！水多了，“河”里的船（股票）自然“水涨船高”。这对整个A股（尤其是券商股）是大利好。

**北向资金 (外资)：**

- **啥意思：** 就是通过“沪港通”、“深港通”从香港那边买A股的“外国钱”。
- **为啥重要：** 它们通常被认为是“聪明钱”(Smart Money)。它们往往更看重“基本面”，而且是“中长线”投资。
- **怎么看：** 交易软件里都有“北向资金流向”。如果它们在**“持续净流入”**（连续几天都在买），说明它们看好A股，市场情绪会很受鼓舞。你还可以看它们在“爆买”哪些股票。

**主力资金 (内资)：**

- **啥意思：** 交易软件会帮你估算“大单”、“超大单”（也就是“主力”）是在“净流入”还是“净流出”某只股票或某个板块。
- **怎么看：**
    - 如果一只股票，股价在涨，“主力资金”也在**“持续净流入”**，这就是“价涨量增”的健康状态。
    - 如果股价在涨，但“主力资金”在**“持续净流出”**，你就要万分小心了！这说明“主力”在“拉高出货”，边拉边卖给你，这叫“背离”。

**两融余额 (杠杆资金)：**

- **啥意思：** 就是“融资融券”的钱，代表了市场上“加杠杆”的“投机客”的情绪。
- **怎么看：**
    - **“融资余额”**（借钱买股）**持续上升**：说明“赌徒”们很兴奋，市场人气旺盛，是“加速器”。
    - **“融资余额”**（借钱买股）**持续下降**：说明大家在“去杠杆”跑路，市场情绪悲观。
    - **注意：** 这也是“双刃剑”。涨的时候它助涨，跌的时候它会“平仓”（被券商强行卖掉），导致“踩踏”和“暴跌”。

## 成交量

**“量价齐升”** 和 **“量价背离”**

1、量价配合

- **价涨量增：** 股价上涨，成交量也放大。这说明涨得“很扎实”，大家都在抢着买，好事。
- **价跌量缩：** 股价下跌，成交量也萎缩。这说明跌得“没力气”，持股的人都不愿意卖了，可能快跌到底了，是好事。

2、量价背离

- **价涨量缩：** 股价在涨，成交量反而小了。这说明涨得“很虚”，没人跟风，主力可能在“拉高出货”，**非常危险**。
- **价跌量增：** 股价在跌，成交量反而放大。这说明有“恐慌盘”在不计成本地跑路，**非常危险**，短期别碰。

3、天量

- 在**高位**，突然放出巨大的成交量（比平时高好几倍），但股价不涨了（比如收了十字星或阴线），这大概率是“主力出货”，赶紧跑！
- 在**低位**，长期没人交易，突然放出巨量，股价大涨，这可能是“主力进场”，可以开始关注了。

## 分时线操作

### 如何判断是不是冲高诱多？

- **黄线**是当天的**均价线**，是短线的多空分水岭。
- **白线**是**实时股价**。
- 水下反抽不过黄线，就是诱多，不仅不能买，如果有仓位必须趁反抽卖出。

### 如何判断该票是否是板块内龙头

- 涨停时间
- 身位
- 逆势表现

### 各种形态

**1、分歧**

- **“分歧出妖股”**，只有经过分歧换手（旧的获利盘走了，新的接力资金进来了），股价才能走得更远。但如果分歧后买方输了，就是见顶。
- 成交量大 股价变化大

**2、洗盘**

- **短线要义**：**洗盘必须缩量，且不破关键支撑。**

i） 分时图特征（单日）

- **急跌慢涨**：像跳水一样“刷”一下砸下去（吓死你），然后慢慢悠悠地爬上来。
- **关门打狗**：早盘稍微冲一下，然后全天在水下（绿盘）震荡，让你看着恶心，想割肉。
- **量能萎缩**：**重点！** 下跌的时候，分时成交量是**很小**的（主力没跑，只是散户在吓得乱跑）。
- **支撑强劲**：每次跌到**黄线（均价线）**或者昨日收盘价附近，就跌不下去了，有大单托底。

ii）K线特征（多日）

- **缩量小阴线**：K线是绿的，或者带下影线，但成交量明显比涨的时候小很多。
- **依托均线**：虽然跌，但死死守住5日线或10日线不破。
- **连阳夹阴**：涨三天跌一天，那个跌的一天就是洗盘。

3、出货

分时图特征（单日）

- **钓鱼波**：这是最经典的。股价被直线拉高（很陡峭），然后像鱼竿的线一样，**慢慢悠悠地滑落**，一波比一波低。
- **拉高出货**：早盘快速拉高吸引跟风，然后全天只要有人买，主力就往外抛，不再创新高。
- **对倒放量**：分时图看着红红火火，但量能大得离谱，股价却横着不动。

K线特征（多日）

- **高位放量滞涨**：成交量巨大（天量），但股价就是涨不动，收十字星或者长上影线。
- **大阴线吞噬**：一根大阴线跌穿了过去几天的涨幅（这叫“断头铡刀”）。
- **重心下移**：每天的高点都比前一天低。
