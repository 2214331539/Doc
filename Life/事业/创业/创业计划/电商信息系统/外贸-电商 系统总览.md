一个电商公司每天大概会发生这样的事情：

**商品准备 → 销售预测 → 采购/生产计划 → 库存管理 → 仓储发货 → 订单履约 → 财务结算 → 经营分析**


| 业务环节                | 对应系统                 |
| ------------------- | -------------------- |
| 商品、平台、客户、渠道基础管理     | 平台管理、CRM、DMS、OMS     |
| 销量预测、需求预测           | Forecast 预测          |
| 供应链计划、库存补货、BOM、产能协同 | SCP 供应链              |
| 仓库库存、入库、出库、盘点       | WMS 仓储               |
| 订单处理、履约、售后          | OMS 订单               |
| 生产制造、工单、物料消耗        | MES 制造               |
| 企业总账、采购、库存、财务一体化    | ERP                  |
| 对账、价格、异常、金蝶推送       | EFC 业财协同             |
| 资产、设备、耗材            | EAM 资产               |
| 人员、组织、权限            | 用户管理、部门管理、角色管理、iHR   |
| 数据看板、经营分析           | BI 数据                |
| AI 能力、模型接口          | Thyseed AI、AI Center |
- **业务执行层**：OMS、WMS、MES、EAM、DMS、CRM
- **计划协同层**：Forecast、SCP、ERP、EFC
- **管理支撑层**：BI、用户管理、部门管理、角色管理、平台管理、AI Center、HR

```
销售平台 / 店铺
        ↓
      OMS 订单系统
        ↓
  销量数据、订单数据
        ↓
   Forecast 预测系统
        ↓
   未来需求预测
        ↓
      SCP 供应链计划
        ↓
  补货计划 / 采购计划 / 生产计划
        ↓
 ┌───────────────┬───────────────┐
 ↓               ↓               ↓
ERP            MES             WMS
采购/财务       生产执行         仓储执行
 ↓               ↓               ↓
        EFC 业财协同
        ↓
      金蝶/财务系统
        ↓
       BI 数据看板
```


| 系统       | 输入             | 输出             |
| -------- | -------------- | -------------- |
| OMS      | 平台订单           | 销量、订单、退款、发货需求  |
| Forecast | 历史销量、活动计划      | 未来销量预测         |
| SCP      | 预测销量、库存、BOM、交期 | 补货计划、采购计划、生产计划 |
| ERP      | 采购、库存、财务数据     | 采购单、应收应付、成本    |
| MES      | 生产计划、BOM       | 生产进度、完工数量      |
| WMS      | 入库/出库任务        | 实际库存、发货结果      |
| EFC      | 订单、库存、采购、平台账单  | 对账结果、财务凭证      |
| BI       | 各系统数据          | 看板、报表、分析       |

## 系统汇总

|系统|核心问题|
|---|---|
|Forecast|未来会卖多少？|
|SCP|为了满足未来销售，现在应该补多少、生产多少、采购多少？|
|WMS|货现在在哪里，有多少，怎么入库出库？|
|OMS|客户订单从哪里来，处理到哪一步？|
|ERP|企业资源、采购、库存、财务如何统一管理？|
|MES|工厂现场如何生产？|
|EFC|业务数据如何对账并进入财务？|
|BI|公司经营状况如何被看见？|
|CRM|客户是谁，如何维护？|
|DMS|经销商如何管理？|
|EAM|企业资产设备如何管理？|
|iHR|人员招聘与人力资源如何管理？|
|AI Center|AI能力如何统一接入和管控？|
|Thyseed AI|业务人员如何使用AI提效？|
