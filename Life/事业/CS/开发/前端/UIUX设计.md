# 资源积累
### 🧠 A. UI 风格库（Visual Style Library）

用于解决“长什么样好看”
- 极简风（Apple / Linear 风格）
- 治愈风（云海 / 插画 / Soft UI）
- 科技感（Glassmorphism / Cyber UI）
- 杂志风（排版驱动 / 大标题）
- 卡片瀑布流风（Pinterest / 小红书）

👉 每条记录必须包含：
- 截图
- 色彩（主色 / 辅色）
- 字体风格
- 布局结构
- 动效特点

---

### 🎨 B. 动效库（Motion Library）

用于解决“怎么动才高级”

重点收集：

- 页面滚动动效（Scroll-based animation）
- 卡片 hover 动效
- 3D 视差
- 微交互（button / hover / input）
- loading 动效

推荐来源：

- Awwwards 动效网站
- Dribbble motion 关键词
- Framer examples

---

### 🧩 C. 组件库（Component Library）

用于解决“如何工程化复用 UI”

你应该收集：

- Navbar（毛玻璃 / sticky / shrink）
- Card（博客 / 产品 / 图片）
- Masonry layout
- Carousel
- Modal / Drawer
- Timeline / Archive UI

---

### 🌈 D. 灵感站（Inspiration Feed）

每天刷，但必须结构化记录：

- Figma Community
- Dribbble
- Behance
- Awwwards
- Land-book / Mobbin / One Page Love
